## 🎯 Mootdx特性

### 多市场支持

| 市场 | 代码示例 | 功能支持 |
|-----|---------|---------|
| A股 | 600000 / sh.600000 | 日线、分钟K线、复权 |
| 深股 | 000001 / sz.000001 | 日线、分钟K线、复权 |
| ETF | 588000 / 159915 | 日线、分钟K线 |
| 指数 | 000001 / 399001 | 日线、分钟K线、复权 |
| 港股 | 00700 / 00700.HK | 日线、分钟K线、复权 |
| 北交所 | 830799 / bj.830799 | 日线、分钟K线、复权 |

### 服务器管理

- **多服务器支持**：20个通达信服务器
- **自动测试**：测试所有服务器延迟
- **最优选择**：自动选择延迟最低的服务器
- **配置保存**：保存最优服务器到best_server.json
- **自动更新**：7天后自动重新测试

### 数据质量

- **数据清洗**：完整的异常值处理
- **复权支持**：前复权、后复权、不复权
- **格式统一**：与BaoStock版本完全兼容
- **逻辑检查**：价格逻辑、成交量逻辑验证

## 💡 使用场景

### 推荐使用Mootdx版本，如果：

1. **需要多市场数据**
   - 分析港股
   - 分析ETF
   - 分析指数
   - 分析北交所股票

2. **需要实时数据**
   - 日内交易分析
   - 分钟K线分析
   - 实时行情监控

3. **需要稳定性**
   - 多服务器保障
   - 自动故障切换
   - 最优线路选择

4. **需要丰富功能**
   - 更多市场支持
   - 更灵活的代码格式
   - 更好的用户体验
